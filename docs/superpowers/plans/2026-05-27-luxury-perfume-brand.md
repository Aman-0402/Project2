# Luxury Perfume Brand Website — Master Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a production-ready luxury perfume brand showcase website with admin product management, WhatsApp conversion, and cinematic premium UI.

**Architecture:** Next.js 14 (App Router) frontend with protected /admin routes calls Django REST Framework backend over JWT-authenticated REST APIs. MySQL stores data; Cloudinary stores product images. No customer accounts — admin-only platform.

**Tech Stack:** Next.js 14, Tailwind CSS, Framer Motion, Django 5, DRF, MySQL, Cloudinary, JWT (SimpleJWT), Vercel (frontend deploy), VPS/Render (backend deploy)
